# سیستم حضور و غیاب و کوئیز دانشجویان (PHP & MySQL)

این پروژه یک سیستم مدیریت حضور و غیاب و کوئیز کلاسی است که با استفاده از PHP8 و پایگاه داده MySQL طراحی و پیاده‌سازی شده است.

## ۱. پیش‌نیازها

*   **وب سرور:** Apache یا Nginx (لاراگون یا XAMPP توصیه می‌شود).
*   **PHP:** نسخه 8.0 یا بالاتر.
*   **پایگاه داده:** MySQL.

## ۲. راهنمای نصب و راه‌اندازی (در محیط لاراگون)

1.  **کپی فایل‌ها:** پوشه `attendance_quiz_system` را در مسیر `www` لاراگون کپی کنید.
2.  **ایجاد دیتابیس:**
    *   لاراگون را اجرا کنید و روی دکمه **Database** کلیک کنید.
    *   یک دیتابیس جدید با نام `attendance_quiz_db` ایجاد کنید.
3.  **اجرای اسکریپت SQL:**
    *   فایل `sql/database.sql` را در محیط مدیریت دیتابیس (مانند HeidiSQL یا phpMyAdmin) باز کرده و اجرا کنید تا جداول مورد نیاز ساخته شوند.
4.  **تنظیمات اتصال:**
    *   فایل `includes/config.php` را باز کنید. اگر از تنظیمات پیش‌فرض لاراگون استفاده می‌کنید، نیازی به تغییر نیست.
    ```php
    define('DB_HOST', 'localhost');
    define('DB_NAME', 'attendance_quiz_db');
    define('DB_USER', 'root');
    define('DB_PASS', ''); // در لاراگون معمولاً رمز عبور خالی است
    ```
5.  **ایجاد کاربران پیش‌فرض:**
    *   فایل `sql/seed_admin.php` را در مرورگر اجرا کنید (مثلاً `http://localhost/attendance_quiz_system/sql/seed_admin.php`).
    *   این اسکریپت کاربران زیر را ایجاد می‌کند:
        *   **ادمین:** `admin` / `admin123`
        *   **استاد:** `teacher1` / `teacher123`
        *   **دانشجو:** `student1` / `student123` و `student2` / `student123`

## ۳. راهنمای استفاده

### الف. پنل استاد (Teacher)

1.  با نام کاربری `teacher1` وارد شوید.
2.  **مدیریت دروس:** در داشبورد، درس جدید ایجاد کنید (مثلاً "آناتومی").
3.  **مدیریت دانشجویان:** روی "مدیریت دانشجویان" کلیک کرده و دانشجویان نمونه (`student1`, `student2`) را به درس اضافه کنید.
4.  **حضور و غیاب:**
    *   روی "جلسات" کلیک کنید و "ایجاد جلسه امروز" را بزنید.
    *   روی "نمایش QR Code" کلیک کنید. این کد را باید دانشجویان اسکن کنند.
5.  **کوئیز:**
    *   ابتدا به صفحه **بانک سوالات** بروید و چند سوال چهارگزینه‌ای اضافه کنید.
    *   در صفحه جلسات، روی "مدیریت کوئیز" کلیک کنید، سوالات مورد نظر را انتخاب و کوئیز را فعال کنید.
6.  **گزارش‌گیری:**
    *   از طریق لینک‌های "گزارش حضور" و "مدیریت کوئیز" می‌توانید لیست حضور و غیاب و نمرات را مشاهده و خروجی CSV بگیرید.

### ب. پنل دانشجو (Student)

1.  با نام کاربری `student1` وارد شوید.
2.  **ثبت حضور:** لینک QR Code را که استاد نمایش داده است، در مرورگر باز کنید (یا اسکن کنید). حضور شما ثبت می‌شود.
3.  **شرکت در کوئیز:** اگر کوئیزی فعال باشد، در داشبورد نمایش داده می‌شود و می‌توانید در آن شرکت کنید.

## ۴. ساختار فایل‌ها

```
attendance_quiz_system/
├── admin/
├── assets/
├── includes/
│   └── config.php        # تنظیمات اتصال به دیتابیس و توابع کمکی
├── student/
│   ├── dashboard.php     # داشبورد دانشجو
│   ├── mark_attendance.php # صفحه ثبت حضور پس از اسکن QR
│   └── take_quiz.php     # صفحه شرکت در کوئیز
├── teacher/
│   ├── attendance_report.php # گزارش حضور و غیاب
│   ├── dashboard.php     # داشبورد استاد و مدیریت دروس
│   ├── export_attendance.php # خروجی CSV حضور و غیاب
│   ├── manage_students.php # مدیریت دانشجویان هر درس
│   ├── manage_quiz.php   # تنظیم کوئیز برای جلسه
│   ├── question_bank.php # مدیریت بانک سوالات
│   ├── sessions.php      # مدیریت جلسات درس
│   ├── quiz_report.php   # گزارش نمرات کوئیز
│   └── view_qr.php       # نمایش QR Code
├── sql/
│   ├── database.sql      # اسکریپت ساخت جداول
│   └── seed_admin.php    # اسکریپت ایجاد کاربران پیش‌فرض
├── index.php
├── login.php             # صفحه ورود
└── logout.php            # صفحه خروج
```
